String method - Sử lí chuỗi

Js string method

  • string.length - trả về độ dài của string
  • string.indexOf ('str') - trả về vị trí đầu tiên của str trong string. Trả về -1 nếu không tìm thấy.
  • string.slice (start, end) - cắt chuỗi từ vị trí start đến end
  • string.replace ('str', 'str replace') - thay thế str = str replace trong string
  • string.toUpperCase() - viết hoa tất cả các chữ trong string
  • string.toLowerCase() - viết thường tất cả các chữ trong string
  • string.trim() - loại bỏ khoảng trắng ở đầu và cuối string
  • string.split ('kí tự tách') - tách chuỗi thành array
  • string.charAt (index) - lấy ra kí tự bởi index cho trước. trường hợp index k đúng trả về 1 string rỗng

Array trong Js

Tạo mảng

Array method - Xử lí Array

  • arr.length - trả về số element của mảng
  • arr[index] - lấy ra element của mảng theo index
  • arr.toString() - chuyển từ arr sang string
  • arr.join ('kí tự nối') - nối các element của arr thành 1 chuỗi. các element đc nối bởi kí tự nối
  • arr.pop() - xóa element cuối mảng và trả về element đã xóa
  • arr.shift() - xóa element ở đầu mảng và trả về element đã xóa
  • arr.push (element,...) - thêm một hoặc nhiều element vào cuối arr và trả về số lượng element của arr sau khi thêm
  • arr.unShift (element,...) - thêm một hoặc nhiều element vào đầu arr và trả về số lượng element cảu arr sau khi thêm
  • arr.splice(start, deleteCount) - xóa deleteCount element từ vị trí start của arr
  • arr.splice(start, 0, element,...) - chèn element vào vị trí sau start
  • arr.splice(start, deleteCount, element,...) - thay element vào vị trí element bị xóa
  • arr.concat(arr2, ...) - nối arr2 và arr thành 1 new arr
  • arr.slice(start, end) - cắt lấy toàn bộ hoặc 1 vài element

Array method: Phần 2 (Đọc sau khi học function)

  • arr.forEach( function(){} ) - Lặp qua từng phần tử của mảng. trả về từng phần tử và index
  • every( function(){} ) - Lặp qua từng phần tử của mảng. trả về từng phần tử và index - kiểm tra tất cả các phần tử có thỏa mãn, trả về true/false.
  • some( function (){} ) - Lặp qua từng phần tử của mảng. trả về từng phần tử và index - kiểm tra , trả về true nếu có 1 phần tử trong mảng thỏa mãn.
  • find( function() {} ) - Lặp qua từng phần tử của mảng. trả về từng phần tử và index - kiểm tra, trả về phần tử thỏa mãn.
